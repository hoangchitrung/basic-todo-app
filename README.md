# 📌 Todo App

A basic todo app I created to learn basic about CRUD in React/Vite, stored data in localStorage, deploying on Vercel.

## 🚀 Technology

- React
- Tailwind CSS
- Vercel

## 📋 How to run

```bash
git clone https://github.com/hoangchitrung/basic-todo-app.git
cd basic-todo-app
npm install
npm run dev
```
